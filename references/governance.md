# 公司治理专家（OPC增强版）

## 负责范围

组织形式选择、股权结构、联合创始人安排、代持、技术入股、章程设计、法人人格独立、公私分离、股东借款、注册资本、法定代表人、重大事项书面决策、关联交易、对外担保、股权转让、增资减资、注销与清算。

## 小微企业基础盘提醒

虽然本技能强调 OPC，但治理领域不能只盯着“一人公司特殊问题”。

对小微企业而言，以下内容同样属于基础盘，必须保留：

- 股权比例怎么分
- 联合创始人权责怎么约
- 代持是否要做、怎么做
- 技术入股如何定价和留痕
- 增资、减资、股权转让怎么走
- 法定代表人、执行董事、监事谁来做

**换句话说：OPC 是治理强化场景，不是治理缩减场景。**

## OPC 第一原则：法人人格独立

**股东和公司是两个主体。**

OPC 的有限责任保护，不是“天然免死金牌”，而是以 **财产独立、账目独立、决策留痕** 为前提。

### 一句话理解

- 正常状态：股东通常以认缴出资额为限承担责任
- 高风险状态：一旦发生公私混同，法院可能否认公司独立人格，股东对公司债务承担连带责任

### 公私分离五件事

| 要求 | 具体做法 |
|------|----------|
| 账户分离 | 公户、私户绝不混用 |
| 资金留痕 | 股东与公司之间每笔往来都要有依据、凭证和备注 |
| 账目规范 | 独立账簿、完整凭证、按期报税 |
| 资产边界 | 公司资产不是股东私人资产 |
| 审计证据 | 至少保留能证明财产独立的年度材料 |

## 组织形式先选对

| 形式 | 责任 | 税负逻辑 | 适合谁 |
|------|------|----------|--------|
| 一人有限公司（OPC） | 原则上有限责任 | 企业层 + 股东层分别处理 | 想规范合作、扩大业务、强调风险隔离 |
| 个体工商户 | 经营者通常承担无限责任 | 个人经营所得逻辑 | 体量小、线下业务、对资质要求低 |
| 个人独资企业 | 投资人承担无限责任 | 个人所得逻辑 | 特定经营安排，但风险隔离弱 |

**默认建议：**

如果用户计划做 AI 应用、技术服务、持续签合同、对外收款、未来融资或雇人，通常优先考虑 **一人有限公司**，但前提是愿意做基本治理和公私分离。

## OPC 高发治理问题

### 1. 股东借款

- 股东向公司借钱、公司向股东借钱，都不是不能做
- 但必须写清性质、用途、期限、利息或无息安排、还款路径
- 长期挂账、不留痕、跨年度不处理，往往同时引发 **治理风险 + 税务风险**

### 2. 注册资本

新《公司法》自 **2024-07-01** 起实施后，注册资本实缴与认缴安排更应谨慎。

实操上重点看：

- 不要虚高认缴，避免未来被债权人追打
- 不要为了“看起来轻松”写极长认缴期又完全无计划
- 结合 3 到 5 年业务规模、算力、研发、人力成本判断

### 3. 章程不能照搬网模

OPC 章程至少要对以下内容作出清楚安排：

- 注册资本和出资方式
- 重大事项决策机制
- 分红和资金支取规则
- 股东禁止行为
- 清算和退出流程

### 4. 重大事项要留书面决定

OPC 决策快，不代表可以没有记录。至少以下事项建议形成书面股东决定或内部记录：

- 分红
- 大额借款
- 对外担保
- 重大采购或重大合同
- 注册资本变更
- 法定代表人变更
- 清算和注销

### 5. 股权问题不要“先合作再说”

小微企业非常容易在前期把股权问题当成“以后再谈”，但实际高频争议恰恰来自：

- 联合创始人口头约定股权
- 技术贡献没有估值依据
- 代持没有书面协议
- 退出机制、回购机制、归属机制缺失

如果问题涉及联合创始人、技术入股、顾问股、干股、期权口头承诺，默认应提高治理领域权重。

## 分析框架

### 治理问题五问法

1. 主体到底是谁
2. 钱从哪来，到哪去
3. 有没有书面依据
4. 有没有稳定留痕
5. 如果明天出事，拿什么证明公司和个人是分开的

### 关联交易审查

1. 是否有真实交易背景
2. 定价是否可解释
3. 是否有协议和审批
4. 资金流是否清楚
5. 是否损害公司利益或规避监管

## 高频问题 FAQ

**Q：OPC 最高发的法律风险是什么？**
公私财产混同。对创始人来说，这不是“形式问题”，而是会不会从有限责任掉回无限责任的问题。

**Q：怎么证明公司财产独立？**
看一整套证据链：独立账户、独立账簿、规范合同、付款凭证、税务申报、年度材料、书面决策记录。不是靠一句“我一直分得很清楚”。

**Q：个人先替公司垫钱可以吗？**
可以，但要短期、留痕、可回补。建议写清“股东垫付 / 借款”的性质，付款备注清楚，并尽快用公户回补。

**Q：股东从公司拿钱最安全的方式是什么？**
通常优先走工资、报销、借款、分红等可解释路径，而不是直接从公司转到个人账户后再说。具体路径要和税务一起看。

**Q：联合创始人股权一开始怎么分更稳？**
先看三件事：谁全职投入、谁持续负责关键业务、谁承担主要资源和风险。不要只按“谁最先认识谁”或“口头承诺”分股。至少要同步约定进入条件、退出条件、回购或归属安排。

**Q：技术入股能不能只口头说“算你多少股”？**
不建议。技术入股至少要解决四个问题：技术是什么、如何评估、何时完成交付、对价对应多少股权或多少期权。否则后续容易同时引发治理、税务和合同争议。

**Q：代持是不是只要双方信任就行？**
不行。代持本质上是高风险安排，至少要有书面协议，写清出资、表决、分红、显名化条件、退出与违约责任。没有协议时，后续非常容易失控。

**Q：法定代表人一定要自己当吗？**
不一定，但如果挂名给别人，自己又实际控制，风险和失控成本都可能更高。挂名和被挂名都不是轻松安排。

**Q：认缴出资期写得很长，有问题吗？**
不是越长越安全。认缴安排应与业务规模、偿债能力和公司法下的新制度配套考虑；过长且无计划，未来可能在融资、合作、债务纠纷里变成问题。

**Q：公司注销为什么也会有个人风险？**
如果不依法清算、账册缺失、债权债务没处理清楚，股东和负责人可能继续暴露在责任链上。

## 风险信号速查

| 风险信号 | 风险描述 | 建议处理 |
|----------|----------|----------|
| 🔴 股东个人账户收公司业务款 | 法人人格否认高风险 | 立即停用私户收款 |
| 🔴 公司账和个人账混在一起 | 不能证明财产独立 | 立即补账、补凭证、补流程 |
| 🔴 公司重大事项没有任何书面记录 | 出事后无法举证 | 建立股东决定模板 |
| 🔴 股东借款长期挂账不处理 | 治理风险 + 税务风险叠加 | 在年度内清理并留痕 |
| 🔴 联合创始人股权安排只有口头约定 | 后续控制权和收益权争议高发 | 立即形成书面协议 |
| 🔴 技术入股没有评估、交付和对价约定 | 后续难确认股权基础 | 补齐技术入股协议和留痕 |
| 🔴 代持没有书面协议 | 显名 / 隐名股东关系失控 | 立即补签代持协议 |
| 🔴 章程直接照搬网上模板 | 关键机制缺失 | 重做 OPC 适配版章程 |
| 🔴 注册资本明显脱离业务现实 | 未来偿债和合作风险 | 调整认缴规划 |
| 🟡 法定代表人和实际控制人长期分离 | 内控与责任链不稳 | 明确授权和变更计划 |
| 🟡 关联交易靠口头说法，没有协议 | 事后难解释 | 补签协议并固化付款路径 |

## 升级条件

- 需要处理股东责任、人格否认、重大债务暴露
- 涉及国资、上市公司、融资并购、复杂代持
- 历史工商、税务、清算问题交织
- 已进入执行、破产、重大争议阶段
