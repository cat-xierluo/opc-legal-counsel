# OPC 政策背景与法规

## 一、OPC 概念界定

**OPC（One Person Company）** 在人工智能时代演变为"单人+AI即公司"的新型创业模式。

### 传统定义
- 2013年《英国公司法》首次明确 OPC 概念
- 界定为"由一人创建、具备永久存续与有限责任特征的公司"

### AI 时代演进
- 大模型、生成式AI、自动化编程工具技术成熟
- 个人研发、运营与商业化能力被成倍放大
- "一人+AI"可独立完成从创意到市场的全流程创业闭环

## 三、地方政府政策

### 苏州市政策

**1. 《苏州市人工智能OPC培育发展行动计划（2025-2028年）》
- 制定机构：苏州市政府办公室
- 目标：打造"OPC创业首选城市"

**2. 苏州工业园区《若干措施》**
- 文件：《关于加快打造全球领先的人工智能OPC创业首选区的若干措施》
- 核心：构建全链条支持体系（技术、资金、场景、人才、平台、生态）
- 重点措施：
  - 打造10个以上、超100万平方米的OPC标杆社区
  - 提供租金减免、拎包入驻、零租工位、开放共享等便利
  - 构建10分钟办公生活圈

**3. 各区行动**
| 区域 | 措施 |
|------|------|
| 吴江区 | OPC社区启动 + 苏州大学未来科学与工程学院OPC创新社区 |
| 高新区 | OPC社区暨WaytoAGI苏州AI创客工坊 |
| 相城区 | 首个AIGC视频领域一人公司（苏州画宗人工智能科技） |
| 姑苏区 | 最高200万元应用场景创新奖励（详见 06-姑苏区专项政策.md） |

**4. 服务支撑**
- 苏州市企业综合服务中心"大数据及人工智能（OPC）"服务专窗
- 电话：0512-69820201
- 地址：苏州市姑苏区平泷路251号城市生活广场3楼

**5. 人才政策**
- 2026年苏州创新创业领军人才计划特设人工智能OPC新赛道

**6. 金融支撑**
- 苏创投布局多只基金均可投资OPC项目
- 导向："投早、投小、投硬科技"
- **OPC创业人才贷**（交通银行苏州分行、工商银行苏州分行）：重点聚焦企业人才（实际控制人及核心团队），评价标准为教育背景 + 产业经验，区别于传统信贷模式

### 江苏省省级政策

**《加快构建OPC创新创业生态》（省人社厅提案，2026年）**
- 构建三级普惠算力网络（省级枢纽—区域中心—社区节点）
- 设立省级算力服务补贴专项资金
- 搭建省OPC创新创业平台
- 设立百亿级省级基金
- 2026年省政府工作报告："让'单人成军'蔚然成林"

**省级目标（"十百千万"苏州）**
| 指标 | 目标 |
|------|------|
| OPC社区 | 超30个 |
| 新增培育OPC企业 | 1,000家 |
| OPC人才 | 超10,000名 |
| 目标年份 | 2028年 |

### 沧浪街道（姑苏区）OPC赋能平台

- 大学生创业一次性创业补贴
- 2年社保补贴
- 免费法律咨询（**法律服务入口机会**）
- 财税体检
- 创业大赛推荐

**生态伙伴**：律所（竞争点/合作点）、会计师事务所、街道商会
