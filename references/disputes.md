# 争议解决专家（OPC增强版）

## 负责范围

争议性质判断、证据收集与固定、诉讼时效初筛、诉前催告、和解 / 调解策略、诉讼 / 仲裁路径介绍、财产保全初步判断、执行风险提示、合同争议、侵权投诉、劳动争议、数据 / AI / IP 争议的前置应对。

## 不负责范围

- 不替代正式诉讼 / 仲裁代理
- 不在未看证据原件、合同条款和完整时间线时判断胜诉率
- 不替用户起草正式律师函、起诉状、仲裁申请书或答辩状

## 先补关键事实

至少先补以下 7 个事实：

1. 当前状态是准备沟通、已违约、已投诉、已收到律师函、已立案还是已被起诉
2. 争议类型是什么：合同、付款、IP、数据、劳动、监管、股权还是侵权
3. 有没有合同、订单、聊天记录、邮件、发票、付款流水、验收记录和原始载体
4. 关键时间线是什么：签约、交付、验收、付款、违约、催告、投诉、收函
5. 合同里有没有管辖、仲裁、违约金、解除、通知送达和律师费条款
6. 对方资产、经营状态和履约能力如何
7. 用户希望止损、追款、下架、和解、起诉、应诉还是保全

## 小微企业基础盘提醒

小微企业处理争议最重要的不是一开始就“打官司”，而是：

- 先保全证据和原始载体
- 先把事实时间线做清楚
- 先看合同条款和付款 / 验收触发条件
- 先评估成本、时间、可执行性
- 高风险事件先止损，不要随口承认责任或销毁材料

## 分析方法

### 争议问题六步法

1. **定状态**
   - 规划、催告、投诉、律师函、立案、应诉、执行
2. **定请求**
   - 追款、解除、赔偿、下架、恢复账号、停止侵权、劳动补偿
3. **定证据**
   - 哪些证据已有，哪些缺，哪些需要马上固定
4. **定规则**
   - 合同条款、法律依据、平台规则、监管文书、时效和管辖
5. **定策略**
   - 协商、调解、律师函、保全、诉讼、仲裁、反诉、投诉
6. **定升级**
   - 是否必须外部律师、会计师、安全团队或公证机构介入

### 默认联动领域

- `contracts`：违约、付款、验收、解除、争议解决条款
- `ip`：侵权投诉、素材 / 代码 / 商标 / 商业秘密争议
- `data-compliance`：数据泄露、隐私投诉、用户权利请求
- `regulatory`：监管问询、处罚、平台下架、职业打假
- `employment`：劳动仲裁、离职、竞业、保密争议
- `governance`：公私混同、股东责任、股权和控制权争议

## 证据留痕重点

| 场景 | 最少保留 |
|------|----------|
| 合同追款 | 合同、订单、交付记录、验收确认、付款流水、催告记录 |
| 微信 / 邮件沟通 | 原始载体、完整聊天、对方身份、时间线 |
| AI 输出投诉 | 输入、提示词、输出、模型版本、审核和下架记录 |
| IP 投诉 | 权利凭证、创作过程、授权链路、使用范围 |
| 数据事件 | 日志、影响范围、时间线、通知和处置记录 |
| 劳动争议 | 合同、考勤、工资、社保、通知、交接材料 |

## 高频误区

- 一上来就起诉，不先看证据和可执行性
- 只有截图，没有原始载体
- 收到律师函或监管文书后先删内容、删日志
- 催款只靠语音和电话，没有书面送达
- 和解只口头达成，没有违约责任和付款期限
- 已经超时效或接近时效，还继续拖延

## 风险信号速查

| 风险信号 | 风险描述 | 建议处理 |
|----------|----------|----------|
| 🔴 关键证据只有截图无原始载体 | 证明力不稳 | 立即补录屏、公证或电子存证 |
| 🔴 对方明确拒付且合同验收不清 | 追款难度高 | 先重建交付和验收证据 |
| 🔴 已收到起诉状、仲裁材料或处罚文书 | 程序期限已经启动 | 立即升级律师 |
| 🔴 数据泄露、AI 侵权或监管投诉已公开发酵 | 舆情和监管叠加 | 先止损、封存、统一口径 |
| 🔴 对方资产恶化或转移 | 执行风险高 | 评估财产保全 |
| 🔴 员工 / 顾问带走代码、客户或账号 | IP、数据和用工风险叠加 | 立即回收权限并固定证据 |
| 🟡 催告没有明确金额、期限和后果 | 后续难证明违约扩大 | 补正式催告 |
| 🟡 和解协议没有违约责任 | 对方再违约成本低 | 补违约条款和送达方式 |

## 高频问题 FAQ

**Q：客户欠款，下一步是不是直接起诉？**
不一定。先看合同、验收、付款触发条件和证据链，再决定催告、律师函前置版本、调解、保全或起诉。

**Q：微信聊天记录能当证据吗？**
可以，但要尽量保留原始载体、完整上下文、对方身份和时间线；重要案件可做公证或电子存证。

**Q：收到律师函要不要马上回复？**
先核对对方权利基础、事实和诉求。事实不清时不要急着承认责任，也不要销毁材料。

**Q：和解协议怎么写才有用？**
至少写明争议事实、履行金额 / 行为、期限、违约责任、送达方式和生效条件。

## 升级条件

- 诉讼 / 仲裁已立案，或已经收到法院 / 仲裁委材料
- 争议金额较大、需要财产保全、证据保全或禁令
- 涉及监管、刑事、舆情、群体性纠纷或跨境因素
- 对方已委托律师，或用户准备正式发律师函 / 起诉 / 反诉
