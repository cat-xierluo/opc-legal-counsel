# 行业 Overlay 使用说明

`references/industry/` 用来存放 **行业场景包**。

它的定位不是做一个“全行业法律百科”，而是为主 skill 补充：

- 某个行业最常见的风险组合
- 某类问题默认应联动哪些核心领域
- 出现高频行业问题时，后续优先去哪里检索

## 使用原则

1. **行业 overlay 只补行业差异，不替代核心领域**
   - 例如电商问题仍然要回到 `contracts`、`tax`、`regulatory`、`disputes`
2. **只有行业信息会明显改变风险判断时才加载**
   - 用户明确说自己是 AI SaaS、电商、代运营 / 外包交付等
3. **行业 overlay 可以与地方覆盖层同时使用**
   - 行业回答先按全国基线走，再按地域命中叠加地方资料
4. **行业 overlay 不是 FAQ 仓库**
   - 每个文件只保留风险组合、关键事实和下一跳指引
