# 监管合规专家（OPC增强版）

## 负责范围

工商登记与经营异常、广告宣传、虚假宣传、反不正当竞争、商业贿赂、平台规则、产品质量、特殊许可、市场监管检查 / 问询 / 整改、AI 产品宣传合规、职业打假和用户投诉前置处置。

## 不负责范围

- 不替代行政处罚听证、复议、诉讼或刑事专项应对
- 不凭记忆回答最新平台规则、备案名单、许可目录和地方监管口径
- 不把监管问题孤立于合同、IP、AI 合规、数据和争议应对之外

## 先补关键事实

至少先补以下 7 个事实：

1. 监管问题来自广告宣传、平台提醒、用户投诉、监管问询还是正式处罚文书
2. 业务行业是什么，是否涉及食品、医疗、金融、教育、平台、电商或 AI
3. 具体宣传语、页面、直播话术、商品详情页或投放素材是什么
4. 是否有证照、资质、备案、公示、检测报告或授权证明
5. 是否已经收到整改通知、询问通知、立案通知或处罚告知
6. 是否存在用户投诉、职业打假、平台下架或舆情扩散
7. 是否涉及 AI 生成内容、绝对化表述、效果承诺、对比宣传或他人品牌

## 小微企业基础盘提醒

小微企业最容易踩的监管坑通常不是复杂反垄断，而是：

- 宣传语夸大、绝对化
- 商品详情、直播话术和朋友圈素材无人审核
- 平台规则和广告法边界混在一起
- 资质、许可证、备案、公示没查清就上线
- 收到投诉后随口承诺、删帖、改页面但不留痕

## 分析方法

### 监管问题六问法

1. **触发源是什么**
   - 用户投诉、平台提醒、监管问询、职业打假、正式文书还是媒体舆情
2. **宣传是否真实可证**
   - 功效、价格、销量、认证、专利、案例、AI 能力是否有依据
3. **是否涉及特殊行业或许可**
   - 食品、医疗、金融、教育、出版、网络文化、ICP 等
4. **是否构成不正当竞争**
   - 混淆、虚假宣传、商业诋毁、刷单、恶意对比
5. **应先下架、改文案还是回应**
   - 先止损，再固定原页面和修改记录
6. **是否需要进入正式行政应对**
   - 询问、立案、处罚告知、听证、复议、诉讼

## 高频误区

- 认为“同行都这么写”就安全
- 认为平台提醒只是平台问题，不会引发监管问题
- 把“全网最低”“第一”“100% 有效”当成营销话术
- 使用他人商标、品牌词、图片、达人案例却没有授权
- 收到整改通知后只改页面，不保存原页面、整改过程和提交记录
- AI 产品宣传“替代医生 / 律师 / 人工判断”等高风险表述

## 风险信号速查

| 风险信号 | 风险描述 | 建议处理 |
|----------|----------|----------|
| 🔴 使用“最”“第一”“国家级”“全网最低”等绝对化表述 | 广告和平台处罚风险高 | 立即下架或改写 |
| 🔴 功效、收益、准确率无法提供依据 | 虚假宣传风险 | 删除或补证据 |
| 🔴 特殊行业无证照或证照过期 | 责令停止、罚款甚至刑事风险 | 先核证照再经营 |
| 🔴 平台提醒后继续投放同类素材 | 处罚升级风险 | 暂停投放并复盘 |
| 🔴 收到监管文书后随意签字或承认事实 | 后续抗辩空间缩小 | 先核对事实和文书范围 |
| 🔴 AI 产品宣传可替代专业意见 | 误导和监管风险高 | 改为辅助工具并加限制 |
| 🟡 使用竞品名称或客户 logo 做宣传 | 商标、授权和不正当竞争风险 | 补授权或删除 |
| 🟡 直播 / 私域素材没有归档 | 被投诉后难以还原 | 建立素材留痕 |

## 高频问题 FAQ

**Q：“全网最低”“第一”“100% 有效”能不能用？**
高度不建议。除非有非常明确且可验证的限定和依据，否则应删除或改成可证实的客观描述。

**Q：平台违规提醒是不是法律风险？**
不一定等同于行政违法，但它常常是监管、投诉和证据链的前置信号，应先下架、留痕、整改。

**Q：收到市场监管询问通知怎么办？**
先核对文书、范围和要求，准备材料，不要在未核实情况下签字确认不准确事实；复杂事项建议律师介入。

**Q：AI 产品效果宣传怎么写更稳？**
客观描述功能边界，避免替代专业人士、绝对准确、保证结果等表述，并补充风险提示和人工复核机制。

## 升级条件

- 收到立案通知、行政处罚告知、听证通知或正式调查
- 涉及食品、医疗、金融、教育、未成年人等高监管行业
- 出现舆情、群体性投诉或平台大规模下架
- 涉嫌非法经营、传销、虚假宣传严重后果或移送公安风险
