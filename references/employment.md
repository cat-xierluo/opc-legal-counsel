# 劳动用工专家（OPC增强版）

## 负责范围

劳动关系与劳务关系区分、劳动合同、试用期、工资社保、加班、绩效、调岗调薪、离职、竞业限制、保密义务、兼职 / 顾问 / 外包协作、早期团队搭建和第一个员工合规。

## 不负责范围

- 不替代正式劳动仲裁 / 诉讼代理
- 不在缺少考勤、合同、聊天记录和工资流水时直接判断输赢
- 不把用工问题孤立于 IP、合同、治理和股权激励之外

## 先补关键事实

至少先补以下 7 个事实：

1. 对方身份是什么：员工、兼职、实习生、顾问、外包、联合创始人还是核心工程师
2. 是否固定坐班、接受管理、纳入考核、按月领固定报酬
3. 是否签了劳动合同、劳务合同、顾问协议或外包协议
4. 是否缴纳社保，工资 / 报酬如何发放
5. 是否接触代码、客户资料、商业秘密或核心账号
6. 当前问题在入职、履行、离职、竞业、保密还是争议阶段
7. 是否同时涉及期权、顾问股、技术入股或联合创始人安排

## 小微企业基础盘提醒

OPC 早期可以没有完整人力体系，但不能把所有协作都叫“顾问”来规避劳动法。最容易出事的不是招人本身，而是：

- 名为兼职 / 顾问，实为固定劳动关系
- 第一个员工不签劳动合同、不缴社保
- 外包长期坐班、固定考勤、固定考核
- 核心人员离职没有权限回收和保密交接
- 口头承诺期权、顾问股或奖金，没有书面机制

## 分析方法

### 用工关系六问法

1. **有没有管理从属性**
   - 是否固定坐班、排班、考勤、审批、绩效考核
2. **报酬是不是工资化**
   - 是否按月固定发、是否类似工资条和固定薪酬
3. **工作是否融入公司业务**
   - 是否长期承担核心岗位，而不是独立项目交付
4. **合同名称和实际履行是否一致**
   - 写“顾问 / 劳务”不代表一定不是劳动关系
5. **成果和秘密怎么归属**
   - 代码、文案、客户资源、账号、资料是否有归属和保密条款
6. **退出怎么处理**
   - 离职手续、权限回收、交接、竞业、未结款项和争议证据

## 高频误区

- 以为合同标题写“劳务 / 顾问”就不会形成劳动关系
- 试用期不签合同、不缴社保
- 用“自愿放弃社保”协议替代法定义务
- 离职时只删微信，不做账号、代码仓库和客户资料回收
- 竞业限制只写义务，不按月支付补偿
- 给核心人员口头承诺期权或顾问股，不写归属和回收机制

## 风险信号速查

| 风险信号 | 风险描述 | 建议处理 |
|----------|----------|----------|
| 🔴 入职超过一个月未签劳动合同 | 双倍工资风险 | 立即补签并规范入职材料 |
| 🔴 试用期不缴社保 | 员工可主张权益 | 试用期也按劳动关系处理 |
| 🔴 名义顾问但固定坐班、考勤、考核、月薪 | 事实劳动关系风险高 | 重新设计为劳动合同或真实服务合作 |
| 🔴 口头辞退或微信一句话辞退 | 违法解除风险高 | 走书面程序并固定事实 |
| 🔴 离职未回收账号、代码仓库和客户资料 | 商业秘密和数据风险 | 建立离职交接清单 |
| 🔴 核心人员未签保密和成果归属 | IP 归属和泄密风险 | 补保密、归属和交付条款 |
| 🟡 竞业限制没有补偿金 | 约束力弱，争议风险高 | 按月支付并限定范围 |
| 🟡 外包团队长期融入日常管理 | 用工关系边界模糊 | 改成项目交付和结果验收 |

## 高频问题 FAQ

**Q：写成“兼职顾问协议”就不会是劳动关系吗？**
不会。劳动关系看实际履行，尤其是管理从属、固定报酬、固定时间和是否融入公司业务。

**Q：第一个员工能不能先不签劳动合同？**
不建议。一个月内应签劳动合同；试用期也包含在劳动合同期内。

**Q：外包开发人员做出的代码归谁？**
优先看合同约定。没有约定时，付款不等于当然取得完整权利，应联动 IP 和合同条款处理。

**Q：竞业限制怎么才更稳？**
对象、范围、期限和补偿都要合理。不能把普通员工都无限制纳入，也不能只写义务不付补偿。

## 升级条件

- 劳动仲裁已立案或已经收到仲裁 / 法院材料
- 群体性劳动纠纷、欠薪、社保集中投诉
- 涉及工伤、违法解除、高额赔偿或竞业限制诉讼
- 核心人员离职同时带走代码、客户、数据或商业秘密
