# foundation-04-first-hire-employee-vs-contractor

## 用户问题

"我准备招第一个人，想让对方先以'兼职顾问'的形式帮我做运营，但实际上要固定坐班、固定考核、每月固定报酬。这样能不签劳动合同吗？"

---

## 1. 一句话结论

不能——固定坐班、固定考核、每月固定报酬的组合特征，在司法实践中几乎必然被认定为事实劳动关系，"兼职顾问"的标签无法规避签订劳动合同和缴纳社保的法定义务；不签反而会产生双倍工资、社保补缴和行政处罚风险。

## 2. 涉及领域

- **主领域**：employment（劳动关系认定、劳动合同、社保）
- **联动领域**：governance（公司用工制度基础）

## 3. 关键事实缺口 / 当前假设

| 缺口 | 说明 |
|------|------|
| 对方是否已有全职工作 | 未提及。如果对方已有社保在缴的全职工作，仅影响社保缴纳方式，不影响劳动关系认定 |
| 预计合作时长 | 未提及。短期项目还是长期合作 |
| 是否涉及接触核心业务数据/代码 | 运营岗通常接触客户信息和商业数据，影响保密和竞业安排 |
| 公司是否已有对公账户和社保账户 | 未提及 |

**当前假设**：公司已注册且具备用工条件，对方将是第一个"招进来"的人，实际工作安排为固定坐班、接受管理、按月固定报酬。

## 4. 红线风险与风险等级

| 风险 | 等级 | 说明 |
|------|------|------|
| 事实劳动关系认定 | 🔴 高 | 管理从属性（固定坐班、考核）+ 经济从属性（按月固定报酬）+ 组织从属性（融入公司运营），司法上构成劳动关系的可能性极高 |
| 未签劳动合同的双倍工资风险 | 🔴 高 | 入职超过一个月未签书面劳动合同，员工可主张最长 11 个月的双倍工资差额 |
| 未缴社保的行政处罚和补缴风险 | 🔴 高 | 用人单位不为员工缴纳社保是违法行为，员工随时可向社保机构投诉 |
| 离职时的违法解除风险 | 🔴 高 | 没有劳动合同和规范入职材料，辞退时极易被认定为违法解除 |
| 成果归属和保密风险 | 🟡 中 | 运营岗接触客户资料和商业信息，没有保密和成果归属条款，离职后可能发生争议 |

## 5. 现在就做什么（24 小时内）

1. **放弃"以兼职顾问规避劳动合同"的想法**——实际履行特征已经锁定了劳动关系
2. **选择正确的用工路径**：

   **方案 A：正常签订劳动合同（推荐）**
   - 签订书面劳动合同，约定试用期、岗位、薪酬、工作时间
   - 依法缴纳社保
   - 签署保密协议和成果归属条款
   - 建立规范的入职材料（身份证、银行卡、入职登记表）

   **方案 B：真正的劳务/外包合作（仅在以下条件全部满足时可行）**
   - 对方不接受固定坐班和考勤管理
   - 按项目/成果交付计费，而非按月固定报酬
   - 对方自行决定工作时间和方式
   - 对方可以同时为其他人提供服务
   - 签订明确的劳务协议或外包协议

3. **如果决定走方案 A**，准备以下文件：
   - 劳动合同（含试用期条款、薪资结构、保密和竞业条款）
   - 员工手册或规章制度（至少包含考勤、考核、保密、离职交接）
   - 入职登记表和相关材料

## 6. 接下来怎么做（7 天 / 30 天）

**7 天内：**

- 完成劳动合同签署和入职手续
- 开设公司社保账户（如尚未开设），为员工缴纳社保
- 确认工资发放方式（对公银行转账，备注"工资"）
- 签署保密协议和成果归属条款（运营岗位接触客户数据和商业信息，必须保护）

**30 天内：**

- 完善公司基础用工制度：考勤规则、绩效考核、离职交接流程
- 为员工配备工作设备，明确工作账号权限和离职回收规则
- 如有需要，与律师确认竞业限制条款的合理性（对象、范围、期限、补偿）
- 建立劳动合同到期提醒机制，避免到期后仍在用工但未续签

## 7. 何时必须升级给外部律师 / 会计师 / 专业机构

- 员工已经以"顾问"身份工作了一段时间，现在要转正，需要补签和补缴社保
- 已发生劳动争议或员工威胁投诉
- 涉及工伤认定（没有社保时工伤赔偿全部由公司承担）
- 需要设计复杂的薪酬结构（底薪 + 绩效 + 提成 + 期权）
- 涉及群体性用工（准备同时招多人）

## 8. 法律依据与适用层级

| 依据 | 适用层级 | 说明 |
|------|----------|------|
| 《劳动合同法》第七条 | 全国 | 用人单位自用工之日起即与劳动者建立劳动关系 |
| 《劳动合同法》第十条 | 全国 | 已建立劳动关系，应当自用工之日起一个月内订立书面劳动合同 |
| 《劳动合同法》第八十二条 | 全国 | 超过一个月不满一年未签书面劳动合同，应向劳动者每月支付双倍工资 |
| 《劳动和社会保障部关于确立劳动关系有关事项的通知》（劳社部发〔2005〕12 号） | 全国 | 认定事实劳动关系的三个标准：管理从属性、经济从属性、组织从属性 |
| 《社会保险法》第五十八条 | 全国 | 用人单位应当自用工之日起三十日内为职工办理社保登记 |

> 以上为基于当前假设的法律信息与风险提示，不构成正式法律意见。建议在正式用工前由执业律师确认劳动合同文本和社保安排。
