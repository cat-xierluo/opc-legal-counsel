# foundation-02-first-client-contract

## 用户问题

"我们给第一家企业客户做定制化管理系统，客户发来的合同模板里没有写明确的验收标准、变更流程和尾款触发条件，只写'项目完成后支付剩余 40%'。我应该怎么改？"

---

## 1. 一句话结论

"项目完成后支付剩余 40%"是典型的验收模糊条款，必须补上可量化的验收标准、异议期限、变更确认机制和尾款触发条件，否则交付完成与否全由甲方说了算，你方尾款几乎无法确定触发。

## 2. 涉及领域

- **主领域**：contracts（合同审查、验收标准、付款节点、变更机制）
- **联动领域**：ip（定制化系统的知识产权归属）、contracts-clauses（条款模板参考）

## 3. 关键事实缺口 / 当前假设

| 缺口 | 说明 |
|------|------|
| 定制化管理系统的具体范围 | 交付物清单、功能模块、技术规格是否已有书面确认 |
| 合同总价和付款结构 | 预付款比例、阶段款安排未提及 |
| 是否涉及第三方组件或已有系统对接 | 未提及。影响知识产权条款和交付范围 |
| 管辖和争议解决条款 | 客户模板中是否已有，未提及 |
| 项目工期和里程碑 | 未提及 |

**当前假设**：合同为定制化技术开发服务，总价已初步商定，目前处于合同审阅阶段，尚未签约。

## 4. 红线风险与风险等级

| 风险 | 等级 | 说明 |
|------|------|------|
| 验收标准缺失 | 🔴 高 | 没有客观验收标准，甲方可以无限要求修改或拒绝确认，尾款永远无法触发 |
| 尾款触发条件模糊 | 🔴 高 | "项目完成后"不是法律上的明确条件，全凭甲方主观判断 |
| 没有变更流程 | 🔴 高 | 需求变更在定制开发中几乎必然发生，没有变更机制意味着范围失控、成本无法追加 |
| 知识产权归属未明确 | 🔴 高 | 定制化系统的源代码、部署脚本、文档归属不清，后续可能产生权属争议 |
| 视为验收条款缺失 | 🟡 中 | 没有约定甲方逾期不响应的视为验收通过，甲方可以无限拖延 |
| 违约责任不对等 | 🟡 中 | 客户模板通常对乙方违约写得多、对甲方逾期付款写得少 |

## 5. 现在就做什么（24 小时内）

1. **梳理完整的交付物清单**，包括：功能模块列表、技术规格书、UI 设计稿、源代码、部署文档、使用手册等
2. **拟定验收标准和验收流程条款**，建议修改为：

   ```
   验收标准：以双方书面确认的《需求规格说明书》（附件X）为准。
   验收流程：乙方交付后【5】个工作日内，甲方完成测试并出具书面验收意见；
   逾期未提出书面异议的，视为验收通过。
   ```

3. **补充尾款触发条件**，改为里程碑关联式：

   ```
   合同总价人民币【X】元，分三阶段支付：
   - 签约后3个工作日内支付30%（预付款）
   - 完成核心功能开发并通过阶段性演示后支付30%（里程碑款）
   - 完成全部交付并通过最终验收后10个工作日内支付40%（尾款）
   ```

4. **增加变更流程条款**：

   ```
   任何一方提出需求变更，应以书面形式提出，包括变更内容、影响评估
   （工期和费用）。双方书面确认后作为合同补充。
   未经书面确认的变更，不作为交付义务的一部分。
   ```

5. **补充知识产权归属条款**：明确背景 IP、新开发成果、源代码交付范围

## 6. 接下来怎么做（7 天 / 30 天）

**7 天内：**

- 与客户就修改后的条款逐条沟通确认
- 补充违约责任条款（建议至少包含逾期付款违约金：每日按逾期金额的万分之三至万分之五计算）
- 确认管辖条款（优先选择己方所在地或合同履行地法院）
- 将确认后的需求规格说明书作为合同附件

**30 天内：**

- 建立合同履约留痕机制：每次需求确认、变更、交付都走书面/邮件确认
- 保存所有阶段性交付记录和客户反馈，作为验收证据
- 如有后续二期、三期合作，考虑转为框架合同 + 订单模式

## 7. 何时必须升级给外部律师 / 会计师 / 专业机构

- 合同金额较大（超过 50 万元），或涉及国企、上市公司等高合规要求客户
- 客户不接受任何修改，坚持使用原始模糊条款
- 涉及核心算法、数据接口、第三方系统集成的复杂知识产权安排
- 已发生交付争议、返工纠纷或客户拒绝付款

## 8. 法律依据与适用层级

| 依据 | 适用层级 | 说明 |
|------|----------|------|
| 《民法典》第五百零九条、第五百一十条 | 全国 | 合同履行原则、约定不明时的补充规则 |
| 《民法典》第五百七十七条、第五百八十二条 | 全国 | 违约责任、瑕疵履行救济 |
| 《民法典》第八百五十一条至第八百五十七条 | 全国 | 技术开发合同的成果归属、风险负担和验收规则 |
| 《著作权法》第十九条 | 全国 | 受委托创作的作品，著作权归属由合同约定 |

> 以上为基于当前假设的法律信息与风险提示，不构成正式法律意见。建议在签约前由执业律师审阅最终文本。
