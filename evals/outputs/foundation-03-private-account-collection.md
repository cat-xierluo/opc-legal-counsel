# foundation-03-private-account-collection

## 用户问题

"公司刚成立还没完全走顺，我先用个人微信和银行卡收了几笔客户服务费，之后再转给公司并补开发票，这样处理有没有问题？"

---

## 1. 一句话结论

有问题——私户收公款同时触发公私混同风险（法人人格否认）和税务合规风险（三流不一致），"事后转回+补开发票"不能自动抹掉历史问题，需要立即停用私户收款并逐笔补齐证据链。

## 2. 涉及领域

- **主领域**：governance（公私分离、法人人格独立）
- **联动领域**：tax（三流合一、发票管理、收入确认）

## 3. 关键事实缺口 / 当前假设

| 缺口 | 说明 |
|------|------|
| 公司主体类型 | 未明确。假设为已注册的一人有限公司或有限责任公司 |
| 私户收款笔数和金额 | "几笔"具体多少、金额多大未提及 |
| 收款到转账的时间间隔 | 未提及。间隔越长风险越大 |
| 是否已全部转给公司 | "之后再转"——是否已全部转完未确认 |
| 补开发票时点 | 开票时间是否与收款时间、服务时间匹配 |
| 是否有对应合同和交付记录 | 未提及。影响业务真实性举证 |

**当前假设**：公司已注册为有限责任公司，私户收款金额不大但已发生多笔，部分已转回公司并补开发票，部分尚未完成。

## 4. 红线风险与风险等级

| 风险 | 等级 | 说明 |
|------|------|------|
| 公私财产混同 | 🔴 高 | 股东个人账户收公司业务款是法人人格否认的典型事实，可能导致股东对公司债务承担连带责任 |
| 三流不一致 | 🔴 高 | 合同（公司签）、资金（个人收）、发票（公司开）三者断裂，影响增值税抵扣和企业所得税列支 |
| 收入确认时点问题 | 🔴 高 | 私户收款时间与服务发生时间、开票时间不一致，可能导致纳税义务确认时点错误 |
| 隐匿收入风险 | 🟡 中 | 如果私户收款后未及时入账并申报，可能被认定为隐匿收入 |
| 跨期税务问题 | 🟡 中 | 如果私户收款跨了纳税期或跨了年度，补申报和调整更复杂 |

## 5. 现在就做什么（24 小时内）

1. **立即停用个人微信和银行卡收取任何公司业务款**——所有收入走公司对公账户
2. **逐笔梳理已发生的私户收款记录**：
   - 每笔的收款时间、金额、付款方、对应的合同/订单
   - 哪些已转回公户、哪些还未转
   - 补开发票的时间与收款时间是否匹配
3. **尚未转回公户的款项，尽快转入公司对公账户**，转账备注写明"客户XXX支付XX项目服务费（原由股东代收，现转入公司账户）"
4. **联系财务/代账公司**，逐笔说明情况，确保：
   - 收入已在正确纳税期申报
   - 发票与实际业务对应
   - 记账凭证完整（合同 + 收款流水 + 转账记录 + 发票）

## 6. 接下来怎么做（7 天 / 30 天）

**7 天内：**

- 与代账公司或会计师确认全部私户收款是否已正确入账和申报
- 如有遗漏申报的纳税期，尽快补充申报
- 建立标准化收款流程：客户签约 → 公司对公收款 → 公司开票 → 合同归档
- 所有客户通知新的付款方式（对公转账），不再接受私户付款

**30 天内：**

- 建立完整的公私分离制度：公户和私户严格分开，股东与公司之间每笔资金往来都有书面依据
- 形成股东借款/垫付的规范模板：如确需股东垫付，走书面流程、备注清楚、短期回补
- 至少做一次账务和税务自查，确认历史私户收款不会留下申报缺口
- 建议保留所有梳理记录，作为证明"已主动纠正"的证据

## 7. 何时必须升级给外部律师 / 会计师 / 专业机构

- 私户收款金额较大（累计超过 50 万元）或跨多个年度
- 已收到税务异常预警、稽查通知或税务约谈
- 私户收款期间同时存在公户付款但未开票的情况，可能涉及虚开风险
- 公司已被债权人追债，公私混同可能成为对方主张股东连带责任的依据
- 涉及多个税种的申报调整，需要专业税务师协助

## 8. 法律依据与适用层级

| 依据 | 适用层级 | 说明 |
|------|----------|------|
| 《公司法》第二十三条 | 全国 | 一人有限公司股东不能证明公司财产独立于股东自己财产的，对公司债务承担连带责任 |
| 《税收征收管理法》第六十三条 | 全国 | 纳税人不进行纳税申报、不缴或少缴税款的，构成偷税，面临追缴、滞纳金和罚款 |
| 《发票管理办法》 | 全国 | 发票开具应与实际经营业务情况相符 |
| 国家税务总局关于三流一致的征管要求 | 全国 | 合同流、资金流、发票流应一致并可对应 |

> 以上为基于当前假设的法律信息与风险提示，不构成正式法律意见或税务意见。建议由执业会计师对具体账务进行核验，必要时由律师评估潜在法律风险。
