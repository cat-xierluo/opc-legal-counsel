# foundation-01-founder-equity-oral

## 用户问题

"我和朋友准备一起做一个企业培训 SaaS，我出技术和现有代码，他出客户资源和销售，口头先说 50:50，等业务起来再补协议。这样先干起来可以吗？"

---

## 1. 一句话结论

不建议"先干再说"——口头 50:50 股权约定没有书面协议锁定，一旦产生收益或需要投入更多资源，控制权、退出机制和成果归属都会变成争议高发区；应在正式合作前至少签一份联合创始人协议框架。

## 2. 涉及领域

- **主领域**：governance（股权结构、联合创始人安排、控制权）
- **联动领域**：contracts（联合创始人协议）、tax（技术入股税务处理）、ip（现有代码和后续开发成果归属）

## 3. 关键事实缺口 / 当前假设

| 缺口 | 说明 |
|------|------|
| 是否已注册公司 | 未提及。假设尚未注册 |
| 技术贡献的价值评估 | "现有代码"的商业价值和开发成本未量化 |
| 客户资源能否落地 | "客户资源和销售"是承诺还是已有合同/意向书，不确定 |
| 双方是否全职投入 | 未提及。全职 vs 兼职直接影响股权合理性 |
| 退出机制 | 完全未提及一方中途退出怎么处理 |
| 是否考虑 vesting（归属期） | 未提及 |

**当前假设**：双方尚未注册公司，均未全职投入，未签署任何书面文件。

## 4. 红线风险与风险等级

| 风险 | 等级 | 说明 |
|------|------|------|
| 口头股权约定无法举证 | 🔴 高 | 没有书面协议，后续任何一方都可以否认或重新主张比例 |
| 技术贡献无估值依据 | 🔴 高 | 你出的代码值多少？没有评估，后续既影响股权比例也影响税务处理 |
| 没有退出/回购机制 | 🔴 高 | 一方中途退出或不再投入，另一方无法收回股权，公司决策陷入僵局 |
| 知识产权归属不清 | 🔴 高 | 你带进来的现有代码、后续开发的 SaaS 成果，没有书面约定归谁所有 |
| 50:50 僵局风险 | 🟡 中 | 平均分配股权在决策分歧时没有打破僵局的机制（谁当法定代表人、谁有一票否决权） |
| 税务处理滞后 | 🟡 中 | 技术入股涉及所得税处理，没有提前规划可能产生额外税负 |

## 5. 现在就做什么（24 小时内）

1. **不要急着开始写代码或投入资源**——先完成以下最低限度的书面约定
2. **起草一份联合创始人备忘录（Term Sheet）**，至少写清：
   - 双方出资方式（技术出资 vs 资源出资）和初步估值方法
   - 股权比例（50:50 是否真的合理）和调整机制
   - 谁全职、谁兼职，投入时间预期
   - 退出机制：一方退出时股权如何回购、价格如何确定
   - 知识产权：你带入的现有代码归属、后续开发成果归属
   - 决策机制：日常经营谁拍板，重大事项（融资、增资、出售）需要双方同意
3. **保留现有代码的版本记录和开发日志**，作为你技术贡献的证据

## 6. 接下来怎么做（7 天 / 30 天）

**7 天内：**

- 根据备忘录找律师或使用可靠模板，起草正式的《联合创始人协议》
- 完成公司注册（如尚未注册），在章程中锁定股权结构
- 在协议中约定 vesting 机制（例如 4 年归属期、1 年悬崖期），避免一方拿股后不干活

**30 天内：**

- 对技术出资进行初步评估并形成书面记录（代码清单、功能说明、市场参考）
- 完成知识产权归属协议：你带入的背景 IP 列清、新开发成果归属写明
- 与税务顾问确认技术入股的税务处理路径（是否适用递延纳税政策）
- 建立公司基本账户和财务制度，避免公私混同

## 7. 何时必须升级给外部律师 / 会计师 / 专业机构

- 双方对股权比例、控制权或退出机制无法达成一致，需要专业谈判支持
- 技术出资金额较大（通常超过 50 万元），需要正式评估和税务处理
- 准备引入外部投资人或进行融资
- 一方已经投入大量资源但股权仍未书面确认，争议风险已经出现
- 涉及代持、期权池、员工激励等复杂安排

## 8. 法律依据与适用层级

| 依据 | 适用层级 | 说明 |
|------|----------|------|
| 《公司法》（2024 修订） | 全国 | 股东出资方式、股权比例、股东权利义务、章程必备条款 |
| 《民法典》合同编 | 全国 | 口头协议有效但举证困难；书面合同是证据基础 |
| 《个人所得税法》及财税〔2016〕101 号 | 全国 | 技术入股可能涉及递延纳税优惠政策 |
| 《著作权法》 | 全国 | 软件代码的著作权归属，委托/合作开发成果归属规则 |

> 以上为基于当前假设的法律信息与风险提示，不构成正式法律意见。建议在正式签约前由执业律师结合完整事实复核。
