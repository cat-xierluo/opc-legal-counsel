# foundation-06-overdue-payment-collection

## 用户问题

"客户拖了两个月尾款不付，目前我手里有合同、微信聊天、源代码交付记录和对方确认可以上线的消息。下一步怎么追款最稳？"

---

## 1. 一句话结论

你目前的证据条件较好，下一步应按"书面催告 → 明确金额和期限 → 评估是否需要律师函/保全 → 必要时起诉"的路径推进，关键是先发正式催告函把违约事实固定下来，不要继续只靠微信口头催。

## 2. 涉及领域

- **主领域**：disputes（证据固定、催告策略、诉讼/仲裁路径）
- **联动领域**：contracts（合同付款条款、违约责任、验收条件）

## 3. 关键事实缺口 / 当前假设

| 缺口 | 说明 |
|------|------|
| 合同中尾款触发条件怎么写的 | 未提及。是"验收通过后支付"还是"项目完成后支付"或其他 |
| 合同中有没有违约金、管辖和争议解决条款 | 未提及 |
| 尾款金额 | 未提及。金额大小影响追款策略选择 |
| 客户不付款的理由 | 是对交付不满意、资金紧张还是纯粹拖延 |
| 客户当前经营状态和资产情况 | 未提及。影响执行可行性 |
| 是否已过诉讼时效 | "拖了两个月"——假设未过时效（一般诉讼时效为三年） |

**当前假设**：合同有效，尾款触发条件为"项目完成/验收后支付"，你方已完成交付，对方已确认可上线但拖延付款，尾款金额未明确但可确定。

## 4. 红线风险与风险等级

| 风险 | 等级 | 说明 |
|------|------|------|
| 继续只靠微信口头催款 | 🔴 高 | 口头催款没有法律效力，不能中断诉讼时效，也无法证明你方已正式主张权利 |
| 证据原始载体缺失 | 🟡 中 | 微信聊天只有截图没有原始手机载体的，证明力会打折扣 |
| 客户资产恶化或转移 | 🟡 中 | 拖延时间越长，客户资产状况越不可控，执行越难 |
| 合同验收条件模糊 | 🟡 中 | 如果验收条件本身模糊，客户可能以"未验收通过"抗辩 |
| 超过诉讼时效 | 🟢 低 | 目前仅两个月，一般三年时效，但持续拖延需要注意 |

## 5. 现在就做什么（24 小时内）

1. **固定现有证据**
   - 微信聊天记录：保留原始手机，不要删除，做一次录屏（从打开微信到完整聊天记录逐条滑动）
   - 源代码交付记录：整理交付时间、方式（邮件/网盘/Git）、对方确认消息
   - 对方"确认可以上线的消息"：截图保存并在原始载体中保留
   - 合同原件：确认签字盖章完整

2. **制作事实时间线**
   - 签约日期 → 开发/服务周期 → 交付日期 → 对方确认上线日期 → 应付款日期 → 催款记录 → 当前

3. **起草并发出正式书面催告函**（建议通过邮件 + 快递双渠道发送）
   - 催告函核心内容：
     - 写明合同编号和尾款金额
     - 写明你方已完成交付（附交付证据摘要）
     - 写明对方已确认可上线的事实
     - 给出明确的付款期限（建议 7-10 个工作日）
     - 写明逾期不付的后果（将采取法律手段追偿，包括但不限于诉讼/仲裁、主张违约金和律师费）
     - 快递单号保留，签收记录保留

4. **不要在催款过程中承认任何己方违约或交付瑕疵**

## 6. 接下来怎么做（7 天 / 30 天）

**7 天内（催告期内）：**

- 等待对方回复。如果对方回复并给出付款计划，可以协商但必须落实到书面
- 如果对方对交付质量有异议，要求对方书面提出具体问题，不要口头争论
- 微信聊天记录做公证或电子存证（尤其是对方确认可以上线的关键消息）

**30 天内（催告未果时）：**

- 评估是否发律师函（通常比自行催告更有压力）
- 核查合同中的管辖条款（法院还是仲裁）和违约金条款
- 调查客户经营状态：是否在正常经营、有无被执行记录、资产状况
- 如金额较大且客户有转移资产迹象，考虑申请财产保全
- 在诉讼时效内正式提起诉讼或仲裁

## 7. 何时必须升级给外部律师 / 会计师 / 专业机构

- 催告函发出后对方仍不付款，需要律师函或准备诉讼/仲裁
- 客户提出反诉或质量异议，情况复杂化
- 尾款金额较大（超过 20 万元），值得投入律师费追偿
- 客户经营状况恶化，需要紧急申请财产保全
- 已收到客户的律师函或反诉材料
- 合同约定仲裁条款，需要仲裁代理

## 8. 法律依据与适用层级

| 依据 | 适用层级 | 说明 |
|------|----------|------|
| 《民法典》第五百零九条 | 全国 | 合同履行原则：当事人应当按照约定全面履行义务 |
| 《民法典》第五百七十七条 | 全国 | 违约责任：一方不履行合同义务的，应承担继续履行、赔偿损失等违约责任 |
| 《民法典》第五百七十九条 | 全国 | 当事人一方未支付价款、报酬、租金、利息的，对方可以请求其支付 |
| 《民法典》第一百八十八条 | 全国 | 诉讼时效为三年，自权利人知道或应当知道权利受损之日起算 |
| 《民事诉讼法》第一百零三条 | 全国 | 财产保全：利害关系人因情况紧急，可以在提起诉讼前申请保全 |

> 以上为基于当前假设的法律信息与风险提示，不构成正式法律意见。建议在正式发函或起诉前由执业律师审查证据和合同条款。
