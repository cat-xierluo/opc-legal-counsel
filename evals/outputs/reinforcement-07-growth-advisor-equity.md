# 兼职顾问 3%"顾问股"安排的风险与正确做法

## 1. 一句话结论

不能先口头约好再补文件——"顾问股"不是法定概念，兼职顾问给不可回收的现股风险极高，至少要先把法律工具拆清楚、写书面协议、设定归属条件和退出机制，否则控制权、税务和后续融资都会被拖累。

## 2. 涉及领域

`growth-financing`（主）+ `governance`（联动）+ `contracts`（联动）+ `tax`（联动）

## 3. 关键事实缺口 / 当前假设

| 缺口 | 当前假设 | 对结论的影响 |
|------|----------|--------------|
| 公司当前主体类型 | 暂按一人有限公司处理 | 若为个体户或个人独资，根本不存在"给股份"的法律基础 |
| "3% 顾问股"具体指什么 | 用户未明确——是股权转让、增资、期权、收益分成还是口头承诺 | 不同工具的权利、税务和退出完全不同 |
| 顾问提供的具体贡献 | 暂按"介绍客户和行业资源"处理 | 资源型贡献通常不适合直接给现股，更适合阶段性收益安排 |
| 是否已有其他口头承诺 | 暂按暂无其他处理 | 若已有多人口头承诺，股权结构表可能已失控 |
| 预计融资时间线 | 暂按 6-18 个月内可能融资处理 | 融资前股权结构必须清晰可解释，口头承诺会成为尽调障碍 |

## 4. 红线风险与风险等级

| 风险点 | 等级 | 为什么危险 |
|--------|------|------------|
| 口头承诺股权，没有任何书面文件 | 极高 | 后续双方对"到底给了什么""什么时候给""能不能收回"的理解完全不同，争议几乎不可避免 |
| 给兼职顾问直接承诺不可回收的现股 3% | 极高 | 顾问只提供弱资源（介绍客户），却拿到永久性股权；一旦资源断供或关系破裂，股权无法收回，控制权被稀释 |
| "融资前再补文件"的打算 | 高 | 融资尽调时会要求完整的股权结构表和所有历史协议，口头承诺无法解释，可能导致投资人退出或压低估值 |
| 没有退出和回购机制 | 高 | 顾问离职、失联、未达标、关系恶化时，没有任何工具把股权收回 |
| 税务后果未评估 | 中 | 股权转让、期权行权或收益分配各有不同的税务触发节点，不提前设计可能导致意外的税负 |

## 5. 现在就做什么（24 小时内）

1. **停止口头承诺**：不要再对顾问说"给你 3% 股份"这类无法定义权利的表述
2. **先拆清楚"顾问股"到底是什么法律工具**：
   - 如果是让他成为登记股东（现股），需要走股权转让或增资程序，有明确的权利义务，但退出困难
   - 如果是让他未来在满足条件后取得股权（期权/限制性股权），需要设计归属条件和行权机制
   - 如果只是给收益分成（分红权/虚拟股权），不需要变更工商登记，退出灵活
3. **评估顾问贡献是否值得给现股**：资源型兼职顾问（只介绍客户、不全职投入），通常更适合阶段性的收益分成或附条件的期权，而不是直接给不可回收的现股
4. **准备一份书面顾问协议框架**：至少要覆盖——合作内容、对价、权利类型、归属条件、退出机制、保密和竞业条款

## 6. 接下来怎么做（7 天 / 30 天）

**7 天内：**

- **签署书面顾问协议**：明确以下要素——
  - 顾问提供什么：具体的服务内容、考核标准（如介绍客户数量、资源对接效果）
  - 公司给什么：是现股、期权还是收益分成，比例多少
  - 什么时候拿到：是立即生效还是分期归属（建议分期，如按年或按里程碑）
  - 什么情况下收回：未达标、终止合作、失联、违约时的回购机制
  - 保密和竞业条款
- **与税务顾问沟通**：确认所选工具（股权转让/期权/收益分成）的税务后果，不同路径的税负差异可能很大
- **更新股权结构表**：把所有已有的和计划中的权益安排统一列出，确保总数不失控

**30 天内：**

- **把顾问安排纳入公司整体激励框架**：如果未来还有核心员工期权、技术入股或其他顾问安排，需要一个统一的股权结构规划，避免零散承诺互相冲突
- **评估是否需要调整公司章程**：如果涉及新增股东或股权转让，需同步修改章程和工商变更
- **为融资做准备**：确保所有权益安排都有书面文件、可解释的结构和清晰的税务处理路径

## 7. 何时必须升级给外部律师 / 会计师 / 专业机构

- 已有多个口头承诺对象，股权比例合计已接近或超过可接受范围
- 顾问要求直接成为工商登记股东
- 已进入或即将进入正式融资流程
- 涉及技术入股、复杂税务安排或境外架构
- 顾问协议涉及对赌、优先权、否决权等复杂条款

## 8. 法律依据与适用层级

| 法律依据 | 适用层级 | 说明 |
|----------|----------|------|
| 《中华人民共和国公司法》（2024 年修订）第 71 条等 | 全国法 | 股权转让、增资、股东权利和义务 |
| 《中华人民共和国民法典》合同编 | 全国法 | 合同的成立、履行、变更和违约责任 |
| 《中华人民共和国个人所得税法》及实施条例 | 全国法 | 股权转让、期权行权、收益分配的税务处理 |
| 财政部、税务总局关于股权激励的有关规定 | 全国法，需核验最新口径 | 不同激励工具的税务认定和优惠政策 |

> 本回答为顾问级分诊和动作清单，不替代正式法律意见。"顾问股"不是法定概念，具体法律工具的选择应结合公司实际结构、融资计划和税务安排由律师和会计师共同复核。
