# 第三方大模型处理客户数据的合同与流程补全

## 1. 一句话结论

把客户合同、会议纪要和联系人信息发给第三方大模型处理，必须在合同和流程上同时补齐数据处理边界、服务商核验、客户告知和责任分配，否则商业秘密泄露、个人信息违规和客户索赔风险叠加。

## 2. 涉及领域

`data-compliance`（主）+ `ai-compliance`（主）+ `contracts`（联动）+ `ip`（核验）

## 3. 关键事实缺口 / 当前假设

| 缺口 | 当前假设 | 对结论的影响 |
|------|----------|--------------|
| 客户合同和会议纪要中是否包含个人信息 | 暂按可能包含联系人姓名、手机号、邮箱等处理 | 若涉及敏感信息，需更高保护标准和单独同意 |
| 第三方模型的训练、缓存和删除条款 | 尚未核查，暂按可能存在默认训练留用处理 | 若模型方有训练留用，商业秘密和个人信息外泄风险极高 |
| 是否已有客户合同中的数据处理授权条款 | 暂按未明确约定处理 | 若无约定，将客户材料发给第三方可能违反合同义务和保密责任 |
| 数据是否涉及跨境传输 | 暂按中国境内处理 | 若模型服务商在境外或有境外节点，需叠加数据出境评估 |
| 业务是 ToB 内部处理还是对外提供服务 | 暂按 ToB 企业服务处理 | 若对外提供服务，需叠加标识、公示和投诉机制义务 |

## 4. 红线风险与风险等级

| 风险点 | 等级 | 为什么危险 |
|--------|------|------------|
| 客户合同和会议纪要直接发第三方模型，未核验服务商条款 | 高 | 商业秘密可能被模型方缓存、训练或转处理；客户一旦发现，可同时主张违约和侵权 |
| 联系人信息（姓名、手机号）属于个人信息，发给第三方未告知客户 | 高 | 违反个人信息保护法的告知同意要求；客户可向监管投诉 |
| 客户合同未约定可使用第三方模型处理 | 高 | 可能违反保密义务，客户可主张违约并索赔 |
| 服务商合同中缺少数据处理协议（DPA） | 高 | 责任边界不清，出事后难以向模型方追偿 |
| 输出内容中的客户信息未做脱敏和访问控制 | 中 | 内部人员可接触所有客户原始数据，泄露面扩大 |

## 5. 现在就做什么（24 小时内）

1. **立即核查第三方模型/API 服务条款**：重点检查——输入数据是否用于模型训练、是否有缓存和留存、删除机制是否可配置、是否有转处理或跨境安排、企业版与免费版的条款差异
2. **暂停将未经脱敏的客户原始材料直接发给模型**：在确认服务商条款安全之前，至少对联系人信息、金额、敏感条款做脱敏处理
3. **梳理现有客户合同中的保密和数据处理条款**：确认是否已有约定允许使用第三方工具处理客户材料，还是需要客户补充授权
4. **建立客户材料分类清单**：区分哪些是商业秘密、哪些含个人信息、哪些是一般业务信息，分别设定处理标准

## 6. 接下来怎么做（7 天 / 30 天）

**7 天内：**

- **与第三方模型服务商签署数据处理协议（DPA）**或补充条款：明确数据仅用于指令处理、不得用于训练、留存期限、删除要求、安全措施和违约责任
- **在客户合同中补充 AI 工具使用条款**：告知客户使用第三方模型处理的情况、数据类型、处理目的、安全保障措施，并获得客户同意
- **建立内部数据输入规范**：哪些客户材料可以输入模型、哪些必须脱敏、哪些完全不能输入，形成操作指引
- **核验上游模型的备案/登记状态**：确认所用模型是否已在国家网信办备案，保留核验记录

**30 天内：**

- **完善供应商台账**：记录所有第三方模型/API 的数据处理条款、安全措施、DPA 状态
- **建立访问权限和日志机制**：记录谁在什么时间输入了什么类型的客户数据、模型版本和输出内容
- **评估是否需要切换到更安全的模型服务方案**：如本地部署、私有化 API、明确不训练的企业版
- **制定数据泄露应急预案**：一旦模型方发生安全事件，如何通知客户、止损和应对

## 7. 何时必须升级给外部律师 / 会计师 / 专业机构

- 客户已发现数据被第三方模型处理并提出投诉或索赔
- 第三方模型服务商条款确认存在训练留用或跨境传输，但业务已大量使用
- 数据涉及金融、医疗、法律等高敏感行业客户
- 客户合同中包含严格的数据本地化或限制第三方处理条款
- 已收到监管问询或整改通知

## 8. 法律依据与适用层级

| 法律依据 | 适用层级 | 说明 |
|----------|----------|------|
| 《中华人民共和国个人信息保护法》第 21-23 条 | 全国法 | 委托处理个人信息的告知同意、数据处理协议和责任分配要求 |
| 《中华人民共和国民法典》第 501 条、第 509 条 | 全国法 | 合同保密义务和附随义务 |
| 《中华人民共和国反不正当竞争法》第 9 条 | 全国法 | 商业秘密保护，包括客户名单、合同内容等经营信息 |
| 《生成式人工智能服务管理暂行办法》 | 全国法 | 使用生成式 AI 处理数据的基本合规要求 |
| 《中华人民共和国数据安全法》 | 全国法 | 数据安全基本制度 |
| GB/T 35273《信息安全技术 个人信息安全规范》 | 国家标准 | 个人信息委托处理的操作指引 |

> 本回答为顾问级分诊和动作清单，不替代正式法律意见。涉及服务商条款、客户合同和具体数据处理安排，建议由律师结合完整文件复核。
