# foundation-05-brand-code-rights

## 用户问题

"我们的小程序准备上线，名字还没注册商标，代码有一部分是外包开发的，也没单独写知识产权归属。现在先上线再补可以吗？"

---

## 1. 一句话结论

不建议先上线再补——名称未检索商标可能被迫更名，外包代码未约定权属可能导致你方没有完整使用权，这两件事在上线前处理成本远低于上线后补救；至少应先做商标检索、补签外包知识产权条款，再考虑上线。

## 2. 涉及领域

- **主领域**：ip（商标保护、软件著作权、外包成果归属）
- **联动领域**：contracts（外包合同的知识产权条款补充）

## 3. 关键事实缺口 / 当前假设

| 缺口 | 说明 |
|------|------|
| 小程序名称是否已做商标检索 | 未提及。假设未检索 |
| 外包开发的具体范围 | 哪部分代码是外包的、比例多大、是否涉及核心功能 |
| 外包合同中有无任何知识产权相关条款 | 未提及。假设完全没有 |
| 外包是否使用了第三方开源组件或框架 | 未提及。影响代码使用自由度 |
| 小程序是否已上线运营 | "准备上线"——尚未上线 |
| 是否涉及用户数据收集 | 未提及。影响上线前的数据合规准备 |

**当前假设**：小程序即将上线，名称已确定但未做商标检索，外包代码占比不小，外包合同中未约定知识产权归属。

## 4. 红线风险与风险等级

| 风险 | 等级 | 说明 |
|------|------|------|
| 名称商标冲突 | 🔴 高 | 如果名称已被他人在相关类别注册，上线后可能被投诉侵权、被迫更名，前期品牌投入全部浪费 |
| 外包代码权属不清 | 🔴 高 | 没有书面约定的情况下，付款不等于取得代码著作权。外包方可能主张保留权利，或限制你方修改、再开发 |
| 上线后再改名的成本 | 🔴 高 | 用户认知、搜索排名、品牌资产、物料全部推倒重来 |
| 第三方组件许可风险 | 🟡 中 | 外包方如使用了 GPL 等强传染性开源协议，可能影响你的商业使用和闭源权利 |
| 软件著作权登记缺失 | 🟡 中 | 不影响权利本身，但在维权和融资场景中登记证书是重要证据 |

## 5. 现在就做什么（24 小时内）

1. **立即做商标检索**
   - 在国家知识产权局商标局官网或专业商标查询平台检索小程序名称
   - 重点检索第 9 类（软件）、第 35 类（广告/商业管理）、第 42 类（软件开发/SaaS）
   - 如果发现高度近似商标，先评估风险再决定是否改名或申请

2. **联系外包开发方，要求补签知识产权补充协议**，至少写清：

   ```
   （1）乙方交付的全部源代码、目标代码、设计稿、文档的著作权归甲方所有；
   （2）乙方不得将交付成果用于其他项目或提供给第三方；
   （3）乙方保证交付成果不侵犯第三方知识产权；
   （4）乙方应提供使用的第三方组件/开源组件清单及对应许可协议。
   ```

3. **梳理代码资产清单**：哪些是自研、哪些是外包、哪些使用了第三方框架或组件，形成文档记录

## 6. 接下来怎么做（7 天 / 30 天）

**7 天内：**

- 商标检索无冲突的，尽快提交商标注册申请（可自行通过商标局官网或委托代理机构）
- 完成外包知识产权补充协议签署
- 对小程序做一次开源组件扫描（可用开源工具），确认没有意外的 GPL 等强传染性组件
- 如小程序涉及用户注册、数据收集，同步检查隐私政策和数据合规（参见 data-compliance 领域）

**30 天内：**

- 提交软件著作权登记申请（源代码前后各 30 页 + 软件说明书）
- 完善品牌保护：确认小程序名称与公众号、域名、商标是否一致，建立品牌资产清单
- 将商标注册、代码著作权、外包协议归档，建立 IP 资产管理文件
- 对后续所有外包/合作开发，统一使用包含知识产权归属条款的标准合同

## 7. 何时必须升级给外部律师 / 会计师 / 专业机构

- 商标检索发现已有近似或相同商标注册，需要专业判断是否构成冲突
- 外包方拒绝签署知识产权补充协议，或声称拥有代码权利
- 已收到商标侵权投诉或律师函
- 代码涉及核心算法、商业秘密或需要专利保护
- 小程序用户量较大，涉及融资尽调中的 IP 清理

## 8. 法律依据与适用层级

| 依据 | 适用层级 | 说明 |
|------|----------|------|
| 《商标法》第五十七条 | 全国 | 未经商标注册人许可，在同一种或类似商品上使用相同或近似商标，构成侵权 |
| 《著作权法》第十九条 | 全国 | 受委托创作的作品，著作权归属由合同约定；无约定的，著作权属于受托人（即外包方） |
| 《计算机软件保护条例》 | 全国 | 软件著作权的归属、许可和转让规则 |
| 《反不正当竞争法》第九条 | 全国 | 商业秘密保护——核心代码如未公开且有保密措施，可主张商业秘密 |

> 以上为基于当前假设的法律信息与风险提示，不构成正式法律意见。建议在商标注册和知识产权协议签署前由执业律师复核。
