# 常用合规要求速查

> 供快速定位问题方向，不替代正式法源核对。涉及税率、备案名录、监管公告、平台规则等时间敏感事项时，应以最新官方文件为准。

---

## 一、OPC 特别红线

| 红线 | 为什么危险 | 最低动作 |
|------|------------|----------|
| 私户收公款 | 公私混同，可能引发股东连带责任 | 立即改用公户收款 |
| 公司和个人账目不分 | 无法证明财产独立 | 建独立账和凭证体系 |
| 股东长期向公司借款不处理 | 治理风险与税务风险叠加 | 年度内清理并留痕 |
| 重大事项无书面决定 | 出事后难以举证 | 建立股东决定模板 |

## 二、AI 产品合规（2025-2026）

### 必看节点

| 事项 | 时间 |
|------|------|
| 《人工智能生成合成内容标识办法》发布 | 2025-03-14 |
| 配套国标 `GB 45438-2025` 与标识办法同步实施 | 2025-09-01 |
| 国家网信办持续公告生成式 AI 服务备案 / 登记信息 | 2025-2026 持续进行 |

### 对 OPC 最重要的 5 个检查点

- 是否核验所使用模型或能力的备案 / 登记 / 上线信息
- 是否在显著位置公示模型名称、备案号或上线编号
- 是否在用户协议写清 AI 生成、输入输出边界和投诉路径
- 是否对生成内容落实标识要求
- 是否建立投诉举报、下架和留痕机制

## 三、个人信息保护

### 核心原则

| 原则 | 说明 |
|------|------|
| 合法性 | 有合法依据，不靠“默认同意” |
| 最小必要 | 只收与功能直接相关的数据 |
| 目的明确 | 不得超范围使用 |
| 公开透明 | 隐私政策和提示应看得懂、找得到 |
| 安全保护 | 技术和制度都要有 |

### 敏感个人信息

- 生物识别
- 金融账户
- 行踪轨迹
- 医疗健康
- 精准定位
- 未满 14 周岁未成年人的信息

### 事件响应提醒

- 中国法下，个人信息安全事件通常强调 **立即补救、按规定告知和报告**
- 不要机械套用海外的 “72 小时” 口径

## 四、电子商务 / 平台合规

### 平台义务

| 义务 | 内容 |
|------|------|
| 主体登记 | 平台内经营者信息应真实、可查 |
| 规则公示 | 平台规则和重要变更应显著公示 |
| 知识产权保护 | 应有投诉、处理、反馈机制 |
| 消费者保护 | 退款、售后、退货规则需清楚 |
| 信息安全 | 交易和用户信息不得随意删除、篡改 |

### 广告合规

- 不得使用绝对化用语
- 不得虚构原价、虚假折扣
- 不得夸大 AI 能力或承诺“百分百准确”

## 五、劳动法规

### 试用期

| 合同期限 | 试用期上限 |
|----------|-----------|
| 3 个月至 1 年 | 1 个月 |
| 1 年至 3 年 | 2 个月 |
| 3 年以上或无固定期限 | 6 个月 |

### 基础提醒

- 试用期也要依法缴纳社保
- 入职满 1 个月仍未签劳动合同，风险明显上升
- 兼职、劳务、实习不要用“按月固定工资 + 固定坐班”做成事实劳动关系

## 六、网络安全 / 数据安全

### 常见关注点

| 项目 | 关注点 |
|------|--------|
| 等级保护 | 是否属于需要定级备案和测评的系统 |
| 数据分类 | 是否涉及重要数据、敏感个人信息 |
| 第三方服务商 | 是否有 DPA、权限控制和审计安排 |
| 应急响应 | 是否有联系人、流程、留痕机制 |

---

## 七、发票与税务

### 红线不是“税率记忆”，而是真实交易与留痕

高频检查点：

- 是否存在真实交易
- 合同、发票、资金流是否能对应
- 股东借款、关联收付款是否可解释
- 是否按期申报并保留凭证

### 使用提醒

- 税率和优惠政策变化频繁，回答时应核对最新政策
- 本速查表不固定写死时效性强的税率数字，避免误导

---

## 八、知识产权

### 基础保护期

| 类型 | 一般保护期 |
|------|------------|
| 自然人作品著作权 | 作者终生及死后 50 年 |
| 法人作品 / 软件著作权 | 发表后 50 年（按现行规则理解） |
| 注册商标 | 10 年，可续展 |
| 发明专利 | 20 年 |
| 实用新型专利 | 10 年 |
| 外观设计专利 | 15 年 |

### AI 场景特别提醒

- 复杂提示词、创意决策和版本留痕很重要
- 核心算法保护通常要同时看软件著作权、商业秘密和专利路径

---

## 九、诉讼时效与应对

| 场景 | 提醒 |
|------|------|
| 一般民事纠纷 | 通常从知道或应当知道权利受损起算 3 年 |
| 劳动争议 | 仲裁时效、起诉期限要单独核对 |
| 已收到律师函 / 通知 / 起诉材料 | 先保全证据，再判断回应期限 |
| 准备起诉 | 先看证据，再看成本和执行可能性 |
