# 企业法律风险自检清单

> 适合 OPC / 小微企业做季度或年度自查。默认按“先活下来、再规范化”的思路排列。

---

## 一、OPC 特别红线

- [ ] 是否全部经营收支都通过公司公户处理
- [ ] 是否仍存在个人账户代收公司业务款项
- [ ] 是否建立独立账簿、凭证和报销规则
- [ ] 股东与公司之间的借款、垫付款是否都有书面留痕
- [ ] 年度内是否对长期挂账、股东借款、关联付款做过清理
- [ ] 重大事项是否形成书面股东决定或内部记录

---

## 二、AI 产品上线

- [ ] 是否核验所用模型或能力的备案 / 登记 / 上线信息
- [ ] 产品页、功能页或详情页是否显著公示模型名称、备案号或上线编号
- [ ] 用户协议是否明确 AI 生成、输入使用边界、输出限制和投诉路径
- [ ] 是否评估生成内容标识要求
- [ ] 是否建立侵权投诉、内容下架和应急响应流程
- [ ] 是否评估未成年人、高风险行业或误导性使用场景
- [ ] 是否对第三方模型 / SDK 的条款和数据使用方式做过审查

---

## 三、合同管理

### 3.1 合同签订

- [ ] 所有业务往来是否都有书面合同
- [ ] 合同主体信息是否核验
- [ ] 授权代表是否有有效授权
- [ ] 合同版本和附件是否完整归档

### 3.2 合同履行

- [ ] 合同变更是否有书面记录
- [ ] 交付和验收标准是否清楚
- [ ] 付款条件达成是否有证据留存
- [ ] 履行瑕疵是否及时书面通知对方

### 3.3 技术 / AI 合同

- [ ] 是否明确知识产权归属、背景 IP 和改进成果归属
- [ ] 是否明确训练数据、用户数据和输出责任边界
- [ ] 是否明确 AI 生成内容侵权时的责任分担

---

## 四、公司治理

- [ ] 组织形式是否仍适配当前业务阶段
- [ ] 章程是否为定制版，而非直接照搬网模
- [ ] 注册资本、认缴安排是否与现实业务匹配
- [ ] 法定代表人、实际控制人、对外签约人是否边界清晰
- [ ] 关联交易、担保、分红、重大采购是否有内部审批和留痕
- [ ] 注销、退出、股权转让是否有预案

---

## 五、财务税务

- [ ] 发票是否与真实交易一致
- [ ] 合同、发票、资金流能否相互对应
- [ ] 是否存在股东长期借款、个人消费公司报销等情形
- [ ] 税务申报、记账和凭证保留是否规范
- [ ] 是否存在异常发票、失控票或走逃风险供应商

---

## 六、数据合规

- [ ] 隐私政策是否可见、可读、可落地
- [ ] 是否超范围收集个人信息
- [ ] 敏感信息是否有更高强度的提示和保护
- [ ] 是否有删除、注销、查阅、更正的实际处理路径
- [ ] 是否对第三方 SDK / 服务商建立台账
- [ ] 是否建立数据泄露或安全事件应急流程

---

## 七、知识产权

- [ ] 产品上线前是否做过商标、版权、专利等基础检索
- [ ] 核心算法、代码、提示词、素材是否做好版本留痕
- [ ] 第三方模型、开源组件、字体、图片是否核验授权边界
- [ ] 商业秘密是否有物理、技术和制度保护措施

---

## 八、劳动与用工

- [ ] 是否把兼职、劳务、实习错误做成事实劳动关系
- [ ] 劳动合同是否在法定期限内签订
- [ ] 试用期是否依法缴纳社保
- [ ] 竞业限制、保密义务是否有对价和书面约定
- [ ] 离职、交接、权限回收是否形成闭环

---

## 九、广告与监管

- [ ] 宣传语是否存在绝对化、虚假、夸大或“AI 百分百准确”等表述
- [ ] 是否存在无证经营、超范围经营或未续期许可
- [ ] 是否建立监管问询、用户投诉、侵权投诉的应对路径
- [ ] 直播、短视频、社媒内容是否经过基本审核
