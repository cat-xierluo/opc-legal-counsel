# 常用合同条款库

> 收录常见合同条款模板及风险提示，供起草/审查时参考。

---

## 一、合同主体条款

### 1.1 合格主体核查要点

| 核查项 | 自然人 | 法人 |
|--------|--------|------|
| 身份证明 | 身份证姓名与签章一致 | 营业执照 + 法定代表人证明 |
| 资质许可 | 行业特殊资质 | 经营异常/失信查询 |
| 授权代表 | 民事行为能力 | 授权委托书（重大交易） |
| 代理权限 | 口头授权风险高 | 代理人与职务是否匹配 |

### 1.2 主体信息条款模板

```
甲方（公司名称），统一社会信用代码：（代码），注册地址：（地址）
法定代表人：（姓名），职务：（职务）
```

## 二、标的条款

### 2.1 标的描述原则

- **具体化**：品名、规格、型号、数量必须明确
- **可量化**：服务类合同约定量化标准（人天、成果物）
- **唯一性**：避免歧义，同一事物只用一种表述

### 2.2 常见风险

❌ 风险条款：
```
"提供优质服务"
"相关内容"
"相关费用"
```

✅ 改进条款：
```
"提供不低于【XX】Mbps带宽的网络接入服务"
"交付成果包括：（1）UI设计稿源文件；（2）开发文档"
"包括但不限于：方案制定费【X】万元、人力成本【X】万元"
```

## 三、价款条款

### 3.1 付款节点设计

**原则**：付款节点与风险转移匹配

| 交易类型 | 推荐付款节奏 |
|----------|-------------|
| 货物买卖 | 30%预付 + 70%到货验收 |
| 定制服务 | 30%预付 + 40%里程碑 + 30%终验 |
| 长期服务 | 月付/季付，预付不超过1期 |
| 一次性大额 | 分批付款，留足质保金 |

### 3.2 价格调整条款

```
单价为人民币【X】元/【单位】，在合同期内保持不变。
如需调整，需双方协商一致，并以书面补充协议确认。
```

### 3.3 税率条款

```
本合同约定的价格为含税价/不含税价，税率为【X】%。
如国家税率调整，以实际开票税率为准。
```

## 四、履行条款

### 4.1 交付标准

```
交付成果应满足：
（1）符合附件【X】《技术规格书》约定的标准；
（2）通过甲方组织的验收测试；
（3）附带完整的使用说明和技术文档。
```

### 4.2 验收标准（风险高发区）

**常见问题**：验收标准模糊，导致履约争议

❌ 风险条款：
```
"甲方验收合格后视为交付完成"
```

✅ 改进条款：
```
"验收标准：以甲方书面确认的《验收测试报告》为准。
验收流程：乙方交付后【X】日内，甲方完成测试；
逾期未提出书面异议的，视为验收通过。"
```

### 4.3 所有权/风险转移

```
货物所有权自【付款/到货/验收】之日起转移至甲方。
货物损毁灭失风险按照《民法典》第603条执行。
```

## 五、违约条款

### 5.1 违约金设定原则

- 违约金 = 预估损失的 1.2-1.5 倍
- 过低（< 银行同期贷款）：司法可能调整
- 过高（> 损失的 30%）：可能被司法酌减

```
逾期付款违约金：每日按逾期付款金额的【万分之X】计算
逾期交付违约金：每日按合同总价的【万分之X】计算
```

### 5.2 根本违约条款

```
一方有下列情形之一的，守约方有权书面通知解除合同：
（一）履行迟延超过【X】日；
（二）履行不符合合同约定且经催告后【X】日内未补救；
（三）【其他重大违约情形】。
```

## 六、免责条款

### 6.1 不可抗力条款

```
因下列原因导致任一方无法履行本合同义务，双方均不承担违约责任：
（1）自然灾害（地震、洪水、台风等）；
（2）战争、政府行为、社会异常事件；
（3）【双方约定的其他情形】。
主张不可抗力的一方应在【X】日内书面通知对方，并提供证明材料。
```

### 6.2 责任上限条款

```
除故意或重大过失外，任一方因本合同产生的赔偿责任以【合同总价/特定金额】为上限。
```

## 七、知识产权条款

### 7.1 著作权归属（委托开发）

```
合作过程中产生的所有成果物的著作权归【甲方/双方共有】所有。
乙方保留【工具/通用技术】的著作权。
```

### 7.2 侵权保证

```
乙方保证其提供的【产品/服务】不侵犯任何第三方的知识产权。
如因乙方原因导致第三方主张权利，乙方负责解决并承担全部责任。
```

---

## 八、保密条款

### 8.1 保密信息定义

```
保密信息包括但不限于：
（1）双方披露的技术信息、商业计划、财务数据；
（2）标注"保密"或"机密"的文件资料；
（3）双方知悉的对方商业秘密。
```

### 8.2 保密义务

```
双方应对保密信息严格保密，未经对方书面许可，不得向任何第三方披露。
保密义务在本合同终止后【X】年内继续有效。
```

## 九、争议解决条款

### 9.1 仲裁条款（推荐）

```
因本合同引起的或与本合同有关的任何争议，双方应友好协商解决；
协商不成的，提交【仲裁机构名称】仲裁，仲裁地点为【城市】，
仲裁语言为中文，仲裁裁决为终局裁决。
```

### 9.2 诉讼管辖条款

```
因本合同引起的或与本合同有关的任何争议，
双方同意由【被告住所地/合同履行地】人民法院管辖。
```

## 十、终止条款

### 10.1 正常终止

```
本合同期限届满或双方确认完成约定事项，本合同终止。
```

### 10.2 提前终止（任意解除权）

```
甲方有权提前【X】日书面通知乙方解除本合同，
但应支付乙方截至解除之日已完成工作的合理报酬。
```

## 高风险条款警示清单

| 条款类型 | 风险等级 | 常见问题 |
|----------|----------|----------|
| 无限连带担保 | 🔴 高 | 未经审议提供担保 |
| 独家供应/销售 | 🟡 中 | 绑定过死，缺乏灵活性 |
| 自动续约 | 🟡 中 | 届满前未决策导致被动续约 |
| 适用法律境外法 | 🔴 高 | 管辖成本高，适用陌生法律 |
| 最低采购/销售承诺 | 🟡 中 | 市场变化时成为负担 |
| 竞业限制 | 🟡 中 | 范围过宽可能被认定无效 |
